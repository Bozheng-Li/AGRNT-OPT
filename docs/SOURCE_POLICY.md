# Source and translation policy

## Evidence priority

1. MCP resource, registry API, package registry API, or another official structured interface.
2. Official repository, release, license, and documentation.
3. Official marketplace listing.
4. Trustworthy third-party research used only as corroboration.

Marketplace popularity and generated summaries are discovery signals, not proof of functionality or licensing.

## Localization

Every curated entry preserves the original-language name and description and adds a Chinese translation. Translation should explain the actual capability in natural Chinese, preserve product and protocol names, and separate verified facts from interpretation. Installation commands, parameter names, environment variables, and license identifiers are not translated.

## Crawling and redistribution

Connectors must respect access controls, robots rules, rate limits, terms, and licensing. If a market does not expose a permitted structured interface, store only the minimum link and derived metadata needed for discovery until redistribution rights are clear.
